# Lifecycle Hooks Flow

Diagram of the framework‚Äôs lifecycle execution:

```text
Boot -> Lifecycle Hook (beforeSystem) 
     -> Providers 
     -> Database 
     -> Express 
     -> HTTP Server 
     -> Lifecycle Hook (afterSystem)
     -> Triggering shutdown
     -> Lifecycle Hook (shutdownSystem on exit/SIGINT)
```
